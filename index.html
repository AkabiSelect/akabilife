Document Redactor

Strip PII from financial PDFs before you share them · runs entirely in your browser
🛡 True redaction: Each page is rasterized to an image with PII painted over pixel-by-pixel, then a new PDF is built from those images. The original text bytes are physically gone, nothing can be copy-pasted from under a redaction. Everything happens in this browser tab; no uploads. Refresh or close = everything discarded.
⚠ PDF processing libraries didn't load.
This tool needs pdf.js and pdf-lib to work. Possible causes:
  • You're offline or your network blocked the CDN request
  • An ad blocker or browser extension is blocking cdnjs.cloudflare.com or cdn.jsdelivr.net
  • Some browsers block external scripts when the HTML is opened directly from disk (file://)
Try: refresh the page, disable ad blockers for this file, or open in a different browser.
